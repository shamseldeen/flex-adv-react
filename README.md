# فلكس للدعاية والإعلان — Flex for Advertising

موقع شركة فلكس للدعاية والإعلان — مبني بـ React 18 + TypeScript + Vite

## التقنيات
- **React 18** + **TypeScript**
- **Vite** (bundler)
- **Tailwind CSS** + shadcn/ui
- **React Query** (data fetching)
- **Wouter** (routing)
- **i18n** عربي/إنجليزي

## تشغيل المشروع
```bash
pnpm install
pnpm --filter @workspace/flex-adv run dev
```
